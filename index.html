What is Oneofakind™?

Oneofakind™ is an enhanced and customized android firmware based on the Android Open Source Project (AOSP).

The Aim of Oneofakind™?

To improve the Android Open Source Project and provide a lightweight base Android firmware with essential features for Android devices
